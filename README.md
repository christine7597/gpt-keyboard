# GPT Keyboard - AI Keyboard

AI 기반 스마트 키보드 웹 애플리케이션입니다.

## 기능

- ✅ AI 기반 텍스트 제안 (OpenAI GPT API 사용)
- ✅ 기본 자동완성 기능 (API 키 없이도 사용 가능)
- ✅ 실시간 제안 표시
- ✅ API 키 저장 기능

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. (선택사항) OpenAI API 키 입력 및 저장
3. 텍스트 입력 시 자동으로 제안 표시
4. "AI 제안 받기" 버튼으로 더 똑똑한 제안 받기

## API 키 발급

1. https://platform.openai.com 접속
2. 계정 생성 또는 로그인
3. API Keys 메뉴에서 새 API 키 생성
4. 생성된 키를 애플리케이션에 입력

## 기술 스택

- HTML/CSS/JavaScript
- OpenAI GPT API (선택사항)
- Local Storage
